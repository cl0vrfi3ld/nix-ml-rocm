# nix-ml-rocm

ROCm deserves to be treated as a first-class citizen in the nix world, so I'm making my own Cachix cache to help fill in the gaps.

Check us out at `ml-rocm.cachix.org`

## Source `nixpkgs` branches
- `unstable`
- `25.11` (coming soon!)

## Tracked packages

- `torch` (for python 3.11, 3.12, 3.13)
- `torchaudio` (for python 3.11, 3.12, 3.13)
- `torchvision` (for python 3.11, 3.12, 3.13)
